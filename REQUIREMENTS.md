# Overview

A [raycast](https://developers.raycast.com/) extension that allows the user to quickly access specific parts of the Icy Veins website, specifically for World of Warcraft.

## Requirements

- Use the Raycast API
- The command to invoke is `iv`
- You then type either the shorthand version of the class and spec, for example `shadow priest` or `sp`
- Followed by optional `pvp` or `pve` respectively
- Lastly the sub-page such as `gear`
